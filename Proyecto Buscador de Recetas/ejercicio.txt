Crea un buscador  de recetas que permita a los usuarios ingresar un ingrediente y luego muestre una lista de recetas que contengan ese ingrediente. Utiliza la API pública "https://www.themealdb.com/api/json/v1/1/search.php?s=" para obtener las recetas. Muestra la información de cada receta, incluyendo su nombre, ingredientes y las instrucciones de preparación. Agrega estilos CSS para darle una apariencia atractiva al buscador y las recetas.
